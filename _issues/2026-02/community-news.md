## Multiset

Multiset is a feature that has been in development for over a year that allows you to earn subset achievements without requiring a patched ROM. This has been a very highly requested feature for several years. You will need RetroArch 1.22.1+ or the absolute latest version of RALibRetro. This is a bleeding edge feature that requires a complete emulator upgrade as well as the latest cores. Only the latest emulators can talk to our back-end correctly to partake in multiset functionality.

More details and multiset can be found [here](https://retroachievements.org/forums/topic/34021).

## Site Updates

### RAWeb 2026.01.07
Most of this release is behind-the-scenes stuff to prepare for future functionality. Some players may notice the home page sometimes loads significantly faster for them. Otherwise, all new functionality is for developers.

**Highlights for Developers**
- Version history is now viewable for achievements in the management app. The page also features an AutoCR-inspired logic viewer.
- It's now possible to upload banner art to games. Currently, this only appears on mobile. Before jumping in, please wait until there is word on what the policy for this sort of thing will be.

[Full release notes here](https://github.com/RetroAchievements/RAWeb/releases/tag/2026.01.07)

### RAWeb 2026.01.16
**New Features**
- The multiset beta has ended: developers can now enable multiset on any game. Over 100 games will be enabled for multiset today. A team is now working through the whole library to turn compatible games on.
- Banners have arrived on desktop: see here. The art team will be busy filling these out. Additionally, RA has officially partnered with our friends at SteamGridDB. With proper attribution, we're allowed to use any of their stuff if we'd like.
- The Supported Game Files page now understands when multiset is enabled and shows the true list of all hashes compatible with the set you're looking at.
- We've changed our minimum password requirements. That's why you were logged out. Due to a recent wave of stolen user accounts, if your password has ever appeared in a data breach, you'll now see a red banner after logging in asking you to change your password.

**Bug Fixes**
- Subset links now work properly in DMs.
- Smart quotes and smart apostrophes in achievement titles/descriptions are now auto-converted to normal quotes and apostrophes. GTA3 should no longer show question marks in some Linux distros.
- Game pages with many leaderboards should load ~1s faster.

**Highlights for Developers**
- Maintainer unlocks now appear on your developer feed.
- Management app changes:
- We've added a new navigation component to achievements.
- The hashes table has a new 'Linked By' user filter.
- The Most Reported Games table no longer has broken game URLs.
- Most detail/edit pages now have a consistent 'View on Site' button.
- Global search has been improved.
[Full release notes here](https://github.com/RetroAchievements/RAWeb/releases/tag/2026.01.16)

### RAWeb 2026.01.22
**New Features**
- We've developed a new trending games algorithm. You can now toggle between "Trending" and "Popular" on the home page. How the new algorithm works is detailed here.
- A bunch of CSS adjustments have been made to banners: iPad fixes, minimized gradient banding, etc.

**Bug Fixes**
- Multiset-enabled subsets should now more accurately track playtime.
- When paginating on game lists, opening a game page, and pressing the back button, you'll now be taken back to the page you were on instead of someplace else.

**Highlights for Contributors (Management App)**
- The global achievements list now has author and maintainer filters.
- Code note tooltips in the Logic view are now scrollable.
- The global games list now lets you filter games that have or don't have banners.
- The top-right user menu now links to your profile and your list of active claims.New Features
- We've developed a new trending games algorithm. You can now toggle between "Trending" and "Popular" on the home page. How the new algorithm works is detailed here.
